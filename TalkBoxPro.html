TalkBox-Pro

About the Project

Overview
This project contains two applications.
1. The TalkBox app: This app contains several buttons to play the audio files that are alotted to it. It also contains swap buttons to swap between different set of audio files, this allows the user to not only play audio files that he has alotted to the buttons but also play different audio files contained in the system so he has access to the full playlist.
2. The Configuration app: This main use of this app to allot audio files to the buttons in the TalkBox app. This app also has many more features, make sets of all audio files in the playlist and alott these sets to the swap buttons in the TalkBox app. You can also add new audio files to the playlist by going to FILE-> Add new audio files->browse audio file and add. This app also shows the current settings he has in the TalkBox app and also shows the changes he is making in a different list so he can compare them. Once he has made the change he can save the using the save button(.txt file is generated with the new settings) and launch the TalkBox app using the rocket launcher. He can also new buttons in the TalkBox app and remove buttons in the talkbox app using the "+, -" buttons. He can also add buttons with audio files in between buttons or at a specific number/position in the TalkBox app using the number drop down menu. He can also display all audio files in the playlist list and also audio files with specific format only. To do all this there is also a tutorial which will be seen when you run the app for the first time. But, you can also access the tutorial from the help menu, you can also skip the tutorial using the skip button in the tutorial.
