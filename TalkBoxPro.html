About the Project

Overview
This project contains two applications.
1. The Talkbox App
2. The Talkbox Configuration App
. Developed an application for speech impaired people, where the application has a number of different audio files and it plays them when the button is pressed, there can be a number of buttons on the app to play audio files and the buttons can be configured to play different audio files and swap them and much more using the configuration app.
