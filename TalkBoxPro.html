TalkBox-Pro


About the Project

This project contains two applications.



  1. The TalkBox App
    • This is the primary app used by the speech impaired people.
    • There are several buttons here, which play different audio files.
    • There are a number of swap buttons which swap the set of audio files for the buttons.
  2. The TalkBox Configuration App
    • Used to changed the audio files of the buttons and create a default.
    • Used to change the total number of audio buttons on the TalkBox app.
    • Settings can be changed and saved in a text file.


Please checkout the Presentation for more features, screenshots and working of the application
